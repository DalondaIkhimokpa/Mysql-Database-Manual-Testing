E-Commerce Database Test Cases

This project demonstrates practical SQL testing scenarios including data validation, constraint enforcement, and proper error handling in MySQL.

Data Validation Tests

1. Find Products Under $100

Validate price range constraints


          SELECT product_id, name, price 
          FROM products 
          WHERE price < 100;
Expected Result: Should return only products priced below $100 (headphones in sample data).

Example of products under $100

2. Find Customers Without Emails

Check for NULL values in required fields


          SELECT customer_id, name 
          FROM customers 
          WHERE email IS NULL;
Expected Result: Empty result set proving all customers have emails.

Example of customers without emails

Constraint Validation Tests

3. Attempt Invalid Price Update

Test CHECK constraint on product prices


          UPDATE products 
          SET price = -50 
          WHERE product_id = 101;
Expected Result: Error message about check constraint violation.

4. Delete Customer with Orders

Test foreign key constraint


          DELETE FROM customers WHERE customer_id = 1;
Expected Result: Error 1451 about foreign key constraint failure.

Successful Operations

5. Update Customer Email

Valid data modification


          -- Before:
          SELECT * FROM customers WHERE customer_id = 1;
          
          -- Update:
          UPDATE customers 
          SET email = 'alice_new@example.com' 
          WHERE customer_id = 1;
          
          -- After:
          SELECT * FROM customers WHERE customer_id = 1;
Expected Result: Successful update showing old and new email.

Example of customer email update

6. Delete Orphaned Product

Safe deletion with no dependencies


          -- Create test product:
          INSERT INTO products VALUES (104, 'Test Product', 10.99, 0);
          
          -- Delete it:
          DELETE FROM products WHERE product_id = 104;
          
          -- Verify:
          SELECT * FROM products WHERE product_id = 104;
Expected Result: Product created and deleted successfully.

Example of product deletion
